# CONTRIBUTING

Спасибо за интерес к проекту!

Краткие правила контрибьюции:

- Форкните репозиторий.
- Создавайте отдельные ветки для фич/фиксов: `feature/ваша-ветка` или `fix/описание`.
- Пишите понятные сообщения коммитов и открывайте Pull Request с описанием изменений.
- Соблюдайте стиль кода и добавляйте тесты при необходимости.

Этот файл добавлен для демонстрации коммитов и работы с git.
