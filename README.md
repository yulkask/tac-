# Tac-Tickle MO03

Игра "Так-Тикль" - настольная логическая игра для двух игроков.

## Описание проекта

Проект разрабатывается командой:
- Скоромникова Юлия
- Размаева
- Иващенко

## Правила игры

Подробные правила игры описаны в файле [ПРАВИЛА_ИГРЫ.md](ПРАВИЛА_ИГРЫ.md).
